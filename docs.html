Adding and Changing Pages in the Templete

To create a new page, open up the 'SlickUI Website Templete' directory.
Next, create a new HTML file, for example, 'mypage.html'.
Open up your HTML file in a text editor.
Link the 'SlickUI.css' file in your page.
Now that you have linked the SlickUI plugin, you are ready to start adding content to your page.

Implementing JavaScript Plugins to Your Page

If you want to use JavaScript plugins or program your own, here's how:
If you want to use an existing JavaScript plugin, for example jQuery, copy the file in the SlickUI directory.
After you copied your file, for example, 'jquery-3.1.1.min.js' link it to your page.
Now, you can use functions from that library in your page.
